What is HTML?

HTML stand for Hypertext Markup Language.It is the standard language for creating web pages.

  • It describes the structure of a web page.
  • It consists of a series of elements.
  • The elements tell the browser how to display the content.
  • Elements are displayed by tags.

Attributes

Target Attributes Properties
_blank Opens the linked document in the new tab.
_self Opens the linked document in the same tab.
_parent Opens the linked document in the parent frame.
_top Opens the linked document in the full body of the window.
